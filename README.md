# CS1.6 巷战大地图（de_xiangzhan_big）

这是一个为 CS1.6 设计的**大尺寸巷战爆破地图源文件**，强调：

- 大地图（约 8192 x 8192 工作区域）
- 巷战背景（街区、建筑块、掩体点位）
- 好操控（主路宽、转点清晰、出生点分散）

## 地图文件

- `maps/de_xiangzhan_big.map`

## 设计要点

1. **双爆破点布局**：A 点在东侧高台，B 点在西侧高台，均可由中路和侧巷接近。
2. **中路宽走廊**：保证移动顺畅、不易卡位，适合新手快速建立方向感。
3. **多掩体低高度**：中区箱体与街角掩体以低中高度为主，便于压枪与转移。
4. **出生点分离**：T/CT 分别位于地图两端，推进节奏明确。
5. **雾效与环境光**：强化城市巷战氛围，减少远距离“一眼到底”造成的压迫感。

## 编译提示（GoldSrc 常规流程）

可使用 JACK/Hammer + ZHLT/VHLT 工具链编译：

1. 打开 `de_xiangzhan_big.map`
2. 执行 `hlcsg -> hlbsp -> hlvis -> hlrad`
3. 生成 `de_xiangzhan_big.bsp`
4. 放入 `cstrike/maps/` 后启动游戏测试
